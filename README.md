# Francisco Colmenarez · Portfolio Q1 2026

A premium, high-performance portfolio website built with a focus on aesthetics, interactive storytelling, and cross-device responsiveness.

## ✨ Key Features

- **📱 Full Mobile Responsive**: Custom `useIsMobile` implementation ensuring pixel-perfect layouts from mobile phones to 4K displays.
- **🌎 Multi-language Support**: Full English and Spanish localization throughout all sections and case studies.
- **🎨 Modern Aesthetics**: Glassmorphism design system using CSS Variables, Framer Motion for micro-interactions, and high-end 3D-like animations.
- **⚡ Performance First**: Built on Vite with optimized assets and smooth scrolling logic (Lenis).
- **📘 Detailed Case Studies**: Comprehensive walkthroughs for major projects (Busbanz and Asesoría), including user journey optimization and visual transformations.

## 🛠️ Tech Stack

- **Core**: React 19 + TypeScript
- **Build Tool**: Vite 6
- **Animations**: Framer Motion
- **Scrolling**: Lenis (Smooth Scroll)
- **Icons**: Lucide React
- **Internationalization**: Custom Context-based translations system
- **Styling**: Vanilla CSS (Modern 3.0 Variables + Utility classes)

## 📁 Project Structure

- `src/components`: Highly interactive UI modules (Hero, About, StatsOrbit, etc.).
- `src/pages`: Main landing pages and detailed case studies.
- `src/context`: Language and state management.
- `src/hooks`: Global utilities like `useIsMobile`.
- `src/data`: Centralized translation dictionaries (ES/EN).
- `src/assets`: Project media, 3D assets, and video demos.

## 🚀 Getting Started

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the local development server with HMR:
```bash
npm run dev
```

### Build & Deploy

Generate production bundle:
```bash
npm run build
```

Deploy to GitHub Pages:
```bash
npm run deploy
```
